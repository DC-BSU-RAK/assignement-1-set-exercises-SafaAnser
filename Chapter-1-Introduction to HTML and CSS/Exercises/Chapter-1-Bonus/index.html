What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It defines the elements of a web page, such as headings, paragraphs, links, images, and other media, using tags and attributes. HTML provides the skeleton of a webpage, which is then styled with CSS (Cascading Style Sheets) and made interactive with JavaScript.

Types of tags

    3. The < title > tag defines the title of the HTML document. This title is what appears in the browser tab and helps search engines understand the content of the page.

     < head > 
       < title > My Webpage < /title >

  3. Headings

    4. These are heading tags used to define headings on the page. < h1 > is the most important (largest) heading, and < h6 > is the least important (smallest). 

    example: 
     < h1 > Main Heading < /h1 > 
     < h2 > Subheading < /h2 > 
     < h3 > Smaller Subheading< /h3 >
  4. paragraphs

    5. The < p > tag is used to define paragraphs. It automatically adds some space before and after the paragraph to separate it from other content, making the text more readable.

     examlpe: 
     < p > This is a paragraph of text. < /p >
  5. Hyperlink

    6. The < a > tag is used to create hyperlinks. The href attribute specifies the URL or path to which the link points. Links can also open in new windows using the target="_blank" attribute.

     examlpe: 
     < a href="https://www.example.com"> Click here < /a >
  6. images

    7. The < img > tag is used to embed images in a webpage. The src attribute defines the path to the image file, while the alt attribute provides alternative text if the image cannot be displayed. It's essential for accessibility.

     example: 
     < img src="image.jpg" alt="Description of image" >
  7. Unordered list

    8. The < ul > tag is used to create an unordered list, meaning a list without numbers. It is typically used for lists that don't need to be in a specific order.

     example: 
     < ul > 
       < li > Item 1 < /li > 
       < li > Item 2 < /li > 
       < li > Item 3 < /li > 
     < /ul >
  8. Ordered list

    9. The < ol > tag defines an ordered list, which automatically numbers the list items. It is typically used when the sequence or order of the items is important. 

     example: 
     < ol > 
       < li > First item < /li > 
       < li > Second item < /li > 
       < li > third item < /li > 
     < /ol >
  9. List item

    10. The < li > tag defines a list item and must be placed inside either a < ul > or < ol > element. It is used to add items to ordered or unordered lists.

     examlpe: 
     < ul > 
       < li > Item 1 < /li > 
       < li > Item 2 < /li > 
     < /ul >
  10. Division

    11. The < div > tag is a block-level element used to group content for styling purposes or to structure the layout of the page. It has no semantic meaning by itself and is often used with CSS to apply styles.

     example: 
     < div class="container"> 
       < p >This is inside a div element.< /p > 
     < /div >
  11. Span

    12. The < span > tag is an inline element used to group content for styling or manipulation purposes. Unlike < div >, it does not create a new block and is used within other inline elements.

     examlpe: 
     < p >This is a < span style="color:red;"> red text < /span > inside a paragraph.< /p >
  12. Header

    13. The < header > tag is used to define the introductory content or the navigation section of a webpage. It typically contains elements like a logo, main heading, or navigation links.

     example: 
     < header > 
     < h1 > My Personal Blog < /h1 > 
     < p > Your daily dose of tech articles and tutorials < /p > 
     < /header >
  13. Footer

    14. The < footer > element defines the footer of a document or section. It typically contains copyright information, contact details, or other important information related to the website or section. 

     examlpe: 
     < footer > 
      < p > Author: Abcd < /p > 
       < p > 2025 My Website < /p > 
     < /footer >

What is CSS?

CSS (Cascading Style Sheets) is a style sheet language used to describe the presentation and layout of a web page written in HTML or XML. While HTML provides the structure of a webpage, CSS is responsible for how it looks. It controls aspects like colors, fonts, spacing, positioning, and overall design.

Types of rule

  1. Color

    2. The color property in CSS sets the color of the text content within an element. You can specify color values using color names (e.g., red), hex codes (e.g., #FF5733), RGB values (e.g., rgb(255, 0, 0)), or other color formats. This property affects only the text and does not affect the background or borders.

     example: 
     p { 
         color: blue; 
     }
  2. Background color

    3. The background-color property sets the background color of an element. This property is commonly used to make text or images stand out by applying a contrasting color behind the content. You can use any valid color value (e.g., color names, hex, RGB, or HSL). 

     examlpe: 
     div { 
           background-color: lightyellow; 
     }
  3. Font size

    4. The font-size property defines the size of the font for the text inside an element. This property can be set in various units such as pixels (px), ems (em), rems (rem), percentages (%), or points (pt). A larger value results in larger text, while a smaller value results in smaller text. 

     examlpe: 
     h2 { 
          font-size: 28px; 
     }
  4. Margin

    5. The margin property defines the space outside an element. It creates distance between the element and its neighboring elements. You can specify the margin for all sides at once (e.g., margin: 20px;) or set different margins for the top, right, bottom, and left sides individually.

     examlpe: 
     .box { 
            margin: 20px; 
     }
  5. Padding

    6. The padding property defines the space inside an element, between its content and its border. Increasing the padding creates more space between the content and the element's edges. Like margin, padding can be set for all sides or individually for each side (top, right, bottom, left).

     example : 
    .content { 
               padding: 15px; 
      }
  6. Border

    7. The border property is used to add a border around an element. You can define the border's width, style (e.g., solid, dashed), and color. The border property is shorthand for setting the individual border-width, border-style, and border-color. 

    examlpe: 
    .box { 
           border: 3px solid green; 
     }
  7. text-align

    8. The text-align property is used to align text horizontally inside an element. You can align the text to the left, right, center, or justify it (spread across the width of the container). This property does not work with block elements unless they contain inline content like text.

      example: 
     h1 { 
          text-align: center; 
     }
  8. Font family

    9. The font-family property specifies the typeface (or font) for the text content inside an element. It can take multiple font names, allowing you to specify fallback fonts in case the preferred font is unavailable. Fonts can be defined by their name, such as "Arial", "Times New Roman", or generic families like sans-serif and serif.

     exampple: 
    body { 
           font-family: 'Arial', sans-serif; 
     }
  9. Font weight

    10. The font-weight property defines the thickness or boldness of the text. It can be set to predefined values like normal, bold, or specific numeric values (e.g., 100 to 900, where 400 is normal and 700 is bold). 

     examlpe: 
     p { 
         font-weight: bold;
     }
  10. Width and height

    11. The width and height properties define the size of an element. These properties can be set in various units such as pixels (px), percentages (%), and other relative units. For example, setting width: 100% will make an element's width equal to the width of its parent container.

     examlpe: 
     img { 
           width:100%;
           beight: auto; 
     }
